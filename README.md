# Java Practice Programs 

This repository contains a collection of basic to intermediate Java programs for practicing logic and mathematical operations. Each file focuses on a specific number property, series, or mathematical calculation.

## Files Description

| File Name | Description |
| :--- | :--- |
| **`HappyNumber.java`** | Checks if a given number is a Happy Number (a number which eventually reaches 1 when replaced by the sum of the square of each digit). |
| **`IntegerToBinary.java`** | A program to convert a standard decimal integer into its binary representation. |
| **`RevNum.java`** | Program to reverse a given number. |
| **`Sample.java`** | A sample/test Java program used for basic logic testing. |
| **`armstrong_number.java`** | Checks whether a number is an Armstrong number (e.g., 153 = 1³ + 5³ + 3³). |
| **`disarium_number.java`** | Determines whether a number is a Disarium number (where the sum of its digits powered with their respective positions equals the number itself). |
| **`factorial.java`** | Calculates the factorial of a given number. |
| **`fibonacci_series.java`** | Generates and prints the Fibonacci sequence up to a specified number of terms. |
| **`pallindromCheck.java`** | Checks whether a number (or string) is a palindrome (reads the same forwards and backwards). |
| **`perfect_number.java`** | Checks if a number is a Perfect Number (a positive integer that is equal to the sum of its proper divisors, excluding itself). |
| **`prime_num.java`** | Checks whether a given number is a prime number. |
| **`strong_number.java`** | Determines whether a number is a Strong Number (where the sum of the factorials of its digits equals the number itself, e.g., 145). |
| **`summation.java`** | Calculates the sum of a series of numbers or digits. |

## How to Run

1. Make sure you have the Java Development Kit (JDK) installed.
2. Compile any program using `javac`:
   ```bash
   javac FileName.java
   ```
3. Run the compiled Java class:
   ```bash
   java FileName
   ```
